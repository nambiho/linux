

0. 3가지 파일을 열어서 서버명과 서버의 설치 디렉토리 수정
 
 - 계정 : notes
   서버명 : gwpk1
   data 설치위치 : /domino/notesdata
   프로그램 설치위치 : /opt/ibm/domino/notes

NOTES_USER=notes					# AIX user id for this server   
NOTES_PATH=/domino/notesdata  		        	# Path for server data directory	
NOTES_SERVER=gwpk1					# Name of Domino Server
LOTUS_PATH=/opt/ibm/domino/notes 	                # Program folder of server
OUTPUT_LOG=/domino/notesdata/log/$NOTES_SERVER.log	# Output file for server console


1. FTP로 서버에 업로드
 - notes 권한, notes 홈폴더(/home/notes)

2. 텔넷으로 접속 후 권한 변경

 - chmod 755 dommon
 - chmod 755 domstart
 - chmod 755 domstop


3. 수행
 - 텔넷으로 notes로 로그인

 - ./dommon
